# 六到位统计 Six Standard Statistics PRD

## 需求背景

### 痛点
- **问题现象**：管理层需要从宏观角度查看全省各地市、各区县六到位完成率，便于横向对比和重点督促
- **发生频率**：中
- **当前 workaround**：通过Excel汇总报表

### 业务目标
- **量化指标**：地图加载 < 2s，切换地市 < 500ms
- **目标期限**：持续可用

### 涉及系统/模块
- **模块名称**：六到位统计
- **变更类型**：新增
- **对接接口**：暂无（Mock数据）

---

## 用户故事

### 故事1
- **角色**：地市管理员 / 省管理员
- **功能**：查看浙江省地图，按地市颜色深浅直观了解六到位整体完成率
- **收益**：一眼识别落后地市，快速定位重点督促对象
- **验收条件**：地图正确加载，各地市根据完成率显示不同颜色

### 故事2
- **角色**：管理员
- **功能**：点击某个地市，进入该地市下各区县的详细完成率视图
- **收益**：深入分析单个地市内各区县的完成情况
- **验收条件**：点击地市后，地图切换为区县视图，列表切换为区县数据

### 故事3
- **角色**：管理员
- **功能**：点击环形图徽章，查看该维度（客情/方案等）的详细进度
- **收益**：了解各子项完成情况
- **验收条件**：点击环形图弹出详情浮窗

## 页面结构

### 路由信息
- **路由路径** - 类型：文本；必填：是；示例：`/six-standard-statistics`
- **页面标题** - 类型：文本；必填：是；示例：`六到位`
- **访问权限** - 类型：枚举（登录）；描述：地市管理员/省管理员

### 布局结构
- **布局类型** - 类型：全屏地图叠加层
- **区域-全屏地图** - ECharts浙江省地图（地市/区县两级）
- **区域-顶部毛玻璃Header** - 返回按钮 + 标题 + 清单/统计Tab
- **区域-右上角完成率徽章** - 蓝色圆形徽章，显示"全省"或"地市名" + 百分比
- **区域-左上角返回提示** - 区县视图下显示，点击返回省级
- **区域-底部列表区** - 半透明白色面板，显示排名列表（省级）或区县列表（城市级）

## 验收标准

### 正常流程
- [ ] **操作**：打开 `/six-standard-statistics` → **预期**：全屏地图加载，显示浙江省地市视图，右上角显示全省完成率
- [ ] **操作**：查看列表 → **预期**：地市按完成率降序排列，每项含排名+6个环形图
- [ ] **操作**：点击"杭州市"地图区块 → **预期**：地图切换为杭州下辖区县，列表切换为区县数据
- [ ] **操作**：点击列表中的环形图 → **预期**：弹出详情浮窗，显示该维度各子项进度
- [ ] **操作**：点击左上角返回按钮 → **预期**：返回省级视图

### 异常流程
- [ ] **操作**：GeoJSON加载失败 → **预期**：降级使用阿里云DataV地图源
